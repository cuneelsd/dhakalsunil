    1. < html > < / html >

  • Formatting Tags
    1. < em >: Emphasized (italic)
    2. < strong >: Bold
    3. < sup >: Superscript
    4. < sub >: Subscript
    5. < ins >: Inserted text
    6. < del >: Deleted text
    7. < mark >: Mark text
    8. < big >: Increase font size by one conventional unit
    9. < small >: Decrease font size by one conventional unit
    10. < strike >: Strikethrough (not supported in HTML5)
  • Table tags
    1. < table >: Defines a table
    2. < tr >: Defines a row
    3. < th >: Defines Header cell
    4. < td >: Defines a call
    5. < caption >: Table Caption
    6. < colgroup >: Group of columns in a table
    7. < col >: Used with < colgroup > element to specify column properties for each column.
    8. < tbody >: Used to group body content
    9. < thead >: Used to group header content
    10. < tfooter >: Used to group the footer content
  • List tags
    1. < ul >: Unordered list (Circle, disk, square, none)
    2. < ol >: Ordered list (number, alphabet and Roman number)
    3. < dl >: Description List or Definition List
    4. < li >: Specifies a list item
    5. < dl >: Specifies a description list (definition list)
    6. < dt >: Specifies the term in a description list (definition title)
    7. < dd >: Description of term in description list (definition data)
  • 5 types of attributes in < ol > order list tag
    1. Type 1
    2. Type I
    3. Type i
    4. Type A
    5. Type a
  • 4 types of attributes in < ul > unorder list tag
    1. Type disc
    2. Type circle
    3. Type square
    4. Type none
  • Frames Tags
    1. < frameset >
    2. < frame >: Used to divide webpage into various frames (windows). must be used under < frameset > tag.
    3. < iframe >: Used to put another document within current HTML document. It is inline frame. (no < frameset > tag required)
    4. < base >
  • Forms Tags
    1. < form >: Defines an HTML form to enter input from user side.
    2. < input >: Defines an input control
    3. < textarea >: Defines a multi-line input control.
    4. < label >: Defines a label for an input element.
    5. < fieldset >: It groups the related element in a form.
    6. < legend > : It defines a caption for < fieldset > element.
    7. < select >: Define a drop-down list.
    8. < option >: Defines a group of releted options in a drop-down list.
    9. < optgroup >: Defines a group of related options in a drop-down list.
    10. < button >: Defines a clickable button
